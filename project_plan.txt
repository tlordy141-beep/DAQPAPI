#### Project Overview

In this project I want to make a python API for my DT9805 DAQ. This API will let me configure my channels and turn voltage readings in from the channels into useful data. I will then analyse this data using python packages such as matplotlib.

#### Why am I making this API

I have previously used QuickDAQ and had that system working fine with all pressure sensors and temperature sensors. But the QuickDAQ interface is limited and I want a more flexible approach. The main reason I am doing this is to learn.

#### Sensor Details & DAQ Connection

The DT9805 DAQ is connected to my laptop using a USB cable.

I have previously used the QuickDAQ software to configure the channels and have used it successfully to get temperature and pressure readings.

The DAQ will have the following sensors in the following channels below:

| DAQ Channel | Sensor |
| ----------- | -------|
| AI1         | TC1    |
| AI2         | TC2    |
| AI3         | TC3    |
| AI4         | PT1    |
| AI5         | PT2    |
| AI6         | PT3    |
| AI7         | PT4    |

The TCs are all T-Type thermocouples with a voltage range of -1V to +1V.

The PTs are all 4-20mA PTs with a 40bar range (measuring against relative pressure). The 4-20mA ouput is wired through 240ohm shunt resistors, then this voltage is measured across the analogue input channels.
